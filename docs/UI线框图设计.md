# **SilentGuardian (AI 防沉迷) \- MVP 阶段 UI 线框图与交互规范**

**\[To AI Agent\] UI 绘制指令：**

请根据以下文本线框图（Wireframe）实现界面的 UI 布局。

* **风格要求：** 遵循 Material Design 3 规范。使用大圆角（16dp/24dp）、清晰的阴影层级和柔和的色调。  
* **主色调建议：** 护眼绿（Primary: \#4CAF50）或沉稳蓝（Primary: \#2196F3）。  
* **布局方式：** 使用 ConstraintLayout 以保证多屏幕尺寸的自适应。

## **页面 1：权限引导页 (PermissionActivity)**

**场景：** 用户首次打开 App，或者中途核心权限被系统/熊孩子关闭时强制弹出。

**核心目的：** 确保应用具备运行所需的全部底层权限。

**核心目的：** 动态获取应用列表，提供时长设置。

* **列表加载：** 页面打开时显示 Loading，异步获取应用列表后展示。已在 MMKV 中的应用默认勾选。

* **保存按钮：** 点击后，将选中的包名（StringSet）和时长限制（Int）写入 MMKV，并关闭当前页面，返回主页面。
